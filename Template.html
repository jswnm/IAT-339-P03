Navigation Button

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">Navigation</a> 
.button-nav {
  color: #5890FF;
  background: none;
  border: none;
  text-decoration: none;
  font-weight: bold;
  text-transform: uppercase;
  position: relative;
  transition: color 0.3s ease;
  cursor: pointer;
  padding: 0;
}

.button-nav::after {
  content: '';
  position: absolute;
  left: 0;
  bottom: 0;
  width: 0;
  height: 2px;
  background: #5890FF;
  transition: width 0.3s ease;
}

.button-nav:hover::after, .button-nav:focus::after {
  width: 100%;
}

Bulleted Lists

CSS selector:

.bulleted-list

Sample code:

<ul class="bulleted-list"><li>Item 1</li><li>Item 2</li></ul> 
.bulleted-list {
  list-style-type: disc;
  padding-left: 2em;
  color: #7AB0FF;
  text-shadow: 0 0 4px #242424, 0 0 8px #5890FF22;
}

.bulleted-list li {
  margin-bottom: 0.4em;
  position: relative;
}

.bulleted-list li::marker {
  color: #5890FF;
  font-size: 1.1em;
  text-shadow: 0 0 6px #5890FF55;
}
